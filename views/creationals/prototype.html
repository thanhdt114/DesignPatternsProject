Prototype

Định nghĩa

Mẫu Prototype là một mẫu thiết kế thuộc nhóm tạo ra các đối tượng mới bằng cách sao chép các đối tượng hiện có, thay vì tạo mới bằng cách sử dụng constructor. Mục tiêu của mẫu Prototype là tạo ra các đối tượng một cách linh hoạt và hiệu quả, đồng thời giảm tải cho việc tạo mới đối tượng từ đầu.

Trong Node.js, chúng ta có thể triển khai mẫu Prototype bằng cách sử dụng tính năng sao chép đối tượng có sẵn trong JavaScript. Dưới đây là một ví dụ minh họa:

Class diagram

Code ví dụ

            
    // Prototype object
    const carPrototype = {
        name: "Default Car",
        drive() {
            console.log(`Driving ${this.name}`);
        },
        clone() {
            const clonedCar = Object.create(this);
            return clonedCar;
        },
    };
    
    // Concrete object
    const sedanCar = carPrototype.clone();
    sedanCar.name = "Sedan Car";
    sedanCar.drive(); // Output: Driving Sedan Car
    
    // Concrete object
    const suvCar = carPrototype.clone();
    suvCar.name = "SUV Car";
    suvCar.drive(); // Output: Driving SUV Car

Giải thích code

Trong ví dụ trên, chúng ta định nghĩa một đối tượng nguyên mẫu (carPrototype) với các thuộc tính và phương thức chung. Sau đó, chúng ta sử dụng phương thức clone để sao chép đối tượng nguyên mẫu và tạo ra các đối tượng cụ thể (sedanCar và suvCar). Mỗi đối tượng cụ thể có thể được tùy chỉnh thuộc tính riêng của nó (như tên) và gọi phương thức drive để thực hiện hành động tương ứng.

Mẫu Prototype giúp chúng ta tạo ra các đối tượng mới một cách linh hoạt và hiệu quả bằng cách sử dụng các đối tượng đã tồn tại làm nguyên mẫu, thay vì tạo mới từ đầu. Điều này giúp giảm tải và tăng hiệu suất trong quá trình tạo mới đối tượng.
